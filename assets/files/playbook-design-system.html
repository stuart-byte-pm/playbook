Duration tokens

instant
80ms Immediate feedback — hover state colours, button press

140ms Micro-interactions — icon swap, underline reveal, focus ring

220ms Component transitions — card hover lift, accordion open, nav link

320ms Layout shifts — section reveals, modal entry, sidebar slide

Easing curves

standard cubic-bezier(0.4, 0, 0.2, 1) General-purpose. Default for most transitions.
decelerate cubic-bezier(0, 0, 0.2, 1) Elements entering the screen — settles into place.
accelerate cubic-bezier(0.4, 0, 1, 1) Elements leaving the screen — clean exit.
sharp cubic-bezier(0.4, 0, 0.6, 1) Quick state changes where snap is intentional.

Brand concepts

Positioning line

Connecting you to clarity

Three pillars

Clarity
Control
Confidence

The guiding star

The brand mark — derived from the + symbol — is always referenced as "the guiding star" in copy and code comments. Never described as a plain plus symbol.

Copy rules

  • British English throughout
  • Sentence case — never title case
  • No exclamation marks in formal or website copy
  • Oxford comma always: "clarity, control, and confidence"

Named concepts — always capitalised

  • the Playbook Paradox
  • the Capital Governance Diagnostic
  • the Memory Gap
  • the Translation Gap
  • the Decision Gap
  • the Governance Bridge
  • the Institutional Playbook Model

Words to avoid in copy

leverage (as verb) · stakeholder alignment · value-add · bespoke · cutting-edge · world-class · best-in-class · excited · passionate · delighted
